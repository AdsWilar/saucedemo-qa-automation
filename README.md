# SauceDemo QA Automation

Automated QA project using Selenium WebDriver, Java, TestNG, Maven and Page Object Model over SauceDemo.

## Project Goal

This repository is intended to demonstrate a professional QA Automation project structure, including automated test scenarios, documentation, execution evidence and CI/CD integration.

## Technologies

- Java 21
- Selenium WebDriver
- TestNG
- Maven
- Page Object Model
- GitHub Actions
